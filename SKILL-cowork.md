---
name: humancode
description: HumanCode is David Caldicott's compact natural shorthand for compressing complex thought into AI-parseable, human-readable notation. Use to create a .hc.md companion file, encode a concept or system, translate verbose prose, write a voice profile, compress an FAQ/SOP, or teach the Cut It Down practice. Triggers: HumanCode, write in humancode, encode this, compress this, .hc, .hc.md, companion file, voice profile, hc file.
---

ASCII-only symbols: = + - > < -> ^ @ / : () ; _ [] . — never ∴ Σ ≈ ∫ → −.

Moves: PascalCase nouns (GoodSleep); intent labels with colons (Math:, Reason:); semicolon chains; parens hold natural-English context only (no method calls, no empty parens); ALL_CAPS connector prepositions (_ON_, _THAN_); arrows (->); units attached (£2k, /m, 10kg).

Principle: average human types it quickly and reads it without translation.

Reference + 20 worked examples: github.com/davidshineoncleans/shine-os-humancode
